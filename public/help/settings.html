Settings

markdown version · view in Roam Research

  • All of Roam's settings live in one panel. Open it from the ⋯ menu in the top right, or run Open settings from the Command Palette.
  • The palette also has Settings: [section name] commands that jump straight to a specific section.
  • Two kinds of settings live here. Preferences and Hotkeys are personal: they only change Roam for you. The rest belong to the graph, and on shared graphs most of them need admin rights.

The tabs

  • Preferences: personal display and behavior settings, like brackets around page references, inline reference counts, bullet colors in shared graphs, and experimental toggles such as Better search (Experimental).
  • Sharing: invite people to the graph, set their roles, and control who can export or view history.
  • Attachments: every file uploaded to the graph, in the File Manager.
  • Hotkeys: view and change keyboard shortcuts. See Customizable Hotkeys.
  • Graph: settings for the whole graph, including Graph History, the Graph Homepage, API tokens, graph size, automatic cleanup of empty daily notes, Semantic search (Experimental) embeddings, and Delete Graph.
  • Encryption: change the password of an encrypted graph. This tab only appears on Encrypted Graphs.
  • Account: your email, password, and Delete Account.
  • Data & Backups: automatic backups, import and export, moving attachments between graphs, and Restore Graph.
  • Roam Depot: the extension marketplace and settings for your installed extensions.

Good to know

  • Installed extensions can add tabs of their own to the panel.
  • Some sections only show for graph admins, so your panel may look shorter on a graph you don't own.
