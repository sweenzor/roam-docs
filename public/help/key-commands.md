# Key Commands

- Roam is built to be driven from the keyboard. These are the commands worth learning first.
- This page is just the highlights. To browse every shortcut, use the `?` help icon in the top right, and the [[Command Palette]] shows each command's keys next to its name.
- To change a hotkey (or give keys to a command that has none), head to **Settings → Hotkeys**. [[Customizable Hotkeys]] shows how.
  - Running **Edit Hotkey: [command name]** from the [[Command Palette]] jumps straight to that command's binding.

### Writing and formatting

- **Bold**
  - Mac: `Cmd-b`
  - PC: `Ctrl-b`
- **Italics**
  - Mac: `Cmd-i`
  - PC: `Ctrl-i`
- **Highlight**
  - Mac: `Cmd-h`
  - PC: `Ctrl-h`
- **Strikethrough**
  - Mac: `Cmd-y`
  - PC: `Win-y`
- **Headings 1, 2, 3** (`0` removes the heading)
  - Mac: `Cmd-Alt-1` / `Cmd-Alt-2` / `Cmd-Alt-3` / `Cmd-Alt-0`
  - PC: `Ctrl-Alt-1` / `Ctrl-Alt-2` / `Ctrl-Alt-3` / `Ctrl-Alt-0`
- **Toggle TODO/DONE**
  - Mac: `Cmd-Enter`
  - PC: `Ctrl-Enter`
- **Insert markdown link** (wraps the selected text)
  - Mac: `Cmd-k`
  - PC: `Ctrl-k`
- **Comment on current block**
  - Mac: `Cmd-Alt-=`
  - PC: `Ctrl-Alt-=`

### Working with blocks

- **Indent block** `Tab`
- **Unindent block** `Shift-Tab`
- **Move block up / down**
  - Mac: `Cmd-Shift-↑` / `Cmd-Shift-↓`
  - PC: `Alt-Shift-↑` / `Alt-Shift-↓`
- **Collapse / expand children**
  - Mac: `Cmd-↑` / `Cmd-↓`
  - PC: `Ctrl-↑` / `Ctrl-↓`
- **Zoom in / out of a block**
  - Mac: `Cmd-.` / `Cmd-,`
  - PC: `Alt-→` / `Alt-←`
- **Copy [block ref]([[Block References]])**
  - Mac: `Cmd-Shift-c`
  - PC: `Ctrl-Shift-c`
- **Copy block embed**
  - Mac: `Cmd-Shift-e`
  - PC: `Ctrl-Shift-e`
- **Follow the link under your cursor** `Ctrl-o`
- **Open the link under your cursor in the sidebar** `Ctrl-Shift-o`

### Getting around

- **Search** (see [[Find or create page]] and [[Advanced Search]])
  - Mac: `Cmd-u`
  - PC: `Ctrl-u`
- **[[Command Palette]]**
  - Mac: `Cmd-p`
  - PC: `Ctrl-p`
- **[[Global search]]**
  - Mac: `Cmd-Shift-u`
  - PC: `Ctrl-Shift-u`
- **Open [[Daily Notes]]**
  - Mac: `Ctrl-Shift-d`
  - PC: `Alt-d`
- **Go to previous / next day** `Ctrl-Alt-p` / `Ctrl-Alt-n`
- **Toggle [[Left Sidebar]]**
  - Mac: `Cmd-\`
  - PC: `Ctrl-\`
- **Toggle [[Right Sidebar]]**
  - Mac: `Cmd-/`
  - PC: `Ctrl-/`
- **Go to the last block on the page**
  - Mac: `Cmd-Shift-Enter`
  - PC: `Ctrl-Shift-Enter`
- **[[Zen Mode]]**
  - Mac: `Cmd-k` then `z`
  - PC: `Ctrl-k` then `z`
- **Help menu** `Alt-Shift-h` (the same shortcut cheat sheet as the `?` help icon in the top right)
